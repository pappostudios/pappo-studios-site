Untitled 3D PVP Multiplayer Game משחק 3D PVP מרובה שחקנים (ללא שם)

Unity C# Multiplayer PVP 3D

A competitive 3D PVP game built in Unity. The core loop is fast-paced player-versus-player combat in a 3D environment, focused on skill expression and replayability. Multiplayer networking is a central challenge — designing for low-latency feel, fair matchmaking, and a responsive combat system that works well across network conditions. משחק 3D תחרותי PVP הנבנה ב-Unity. הלולאה המרכזית היא קרב שחקן-מול-שחקן מהיר בסביבה תלת-ממדית, ממוקד בביטוי מיומנות וכיף חזרתי. רשת מרובי שחקנים היא אתגר מרכזי — עיצוב לתחושת השהייה נמוכה, Matchmaking הוגן ומערכת קרב רספונסיבית שעובדת טוב בכל תנאי רשת.

This project is an opportunity to go deeper into Unity's architecture, multiplayer frameworks, and 3D game feel design — areas I'm actively building expertise in. הפרויקט הזה הוא הזדמנות להעמיק בארכיטקטורת Unity, Frameworks מרובי שחקנים ועיצוב Game Feel תלת-ממדי — תחומים שאני בונה בהם מומחיות באופן אקטיבי.
