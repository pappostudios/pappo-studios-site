Milometri

Milometri — Psychometric Vocabulary App

Flutter Dart Python Education iOS & Android

A mobile vocabulary learning app for students preparing for Israel's psychometric (college admissions) exam. Built entirely solo, from the data pipeline and ML difficulty classification to the Flutter UI and store releases. Milometri uses adaptive learning to surface the words each user finds hardest, in both Hebrew and English. Available on both Google Play and the App Store.

What I Built

Complete Flutter app with clean state management, local data persistence, and a smooth card-based review UI. Supports two word banks (Hebrew + English) with thousands of vocabulary items.

Data Pipeline

Wrote Python scripts (classify_levels.py, rate_difficulty.py) to automatically classify and rank vocabulary words by difficulty using frequency analysis and ML techniques.

Architecture

Clean architecture with separated layers for data, domain, and presentation. Optimised for fast startup and smooth performance on low-end Android devices.

Deployment

Published to both Google Play and the App Store using Codemagic CI/CD. Maintained a privacy policy page and handled full store compliance end-to-end.

KJ Fitness App

KJ Fitness — Coaching Mobile App

React Native Expo Firebase Firestore iOS & Android

A full fitness coaching platform built for a real client — KJ Fitness. The app serves two roles: clients see their weekly training plans, track sessions, and pay for sessions; coaches manage client rosters, create weekly content, and monitor progress. Built with React Native + Expo for cross-platform delivery and Firebase for real-time backend.

Dual-role Architecture

Separate screen flows for clients and coaches, unified under a single codebase. Role-based access controlled via Firebase Auth and Firestore security rules.

Client Features

Weekly plan cards, session tracking, in-app payment (Bit Pay integration), quick-action grid, and bottom tab navigation with 5 tabs.

Tech Stack

React Navigation v6, Firebase Firestore + Auth + Storage, Expo managed workflow, custom theme system with design tokens (colors, typography, spacing).

Backend

Firestore with complex security rules ensuring clients only access their own data. Custom indexes for efficient coach-client queries. Firebase Storage for profile media.

KJ Fitness Website

KJ Fitness — Coaching Website

HTML / CSS / JS Firebase Auth Firestore Responsive

The web companion to the KJ Fitness mobile app. A full multi-page coaching website that includes a marketing landing page, a Firebase-authenticated login system, and separate dashboards for clients and coaches. Clients view their plans and progress; coaches manage clients and edit weekly content — all without a traditional backend server.

Pages Built

Landing page (index.html), login page, client dashboard, and coach dashboard — each a fully functional HTML/CSS/JS page.

Auth & Data

Firebase Auth for secure login. Firestore for reading and writing client/coach data live in the browser — no backend server required.

Design

Fully responsive layout. Dark theme with teal accents matching the mobile app's brand. Optimised for both desktop and mobile browsers.

Real Client Delivery

Designed and delivered for KJ Fitness — an actual coaching business. Handled client requirements, branding integration, and production deployment.
