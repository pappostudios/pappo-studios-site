Milometri

Milometri — Psychometric Vocabulary App מילומטרי — אפליקציית אוצר מילים לפסיכומטרי

Flutter Dart Python Education iOS & Android

A mobile vocabulary learning app for students preparing for Israel's psychometric (college admissions) exam. Built entirely solo, from the data pipeline and ML difficulty classification to the Flutter UI and store releases. Milometri uses adaptive learning to surface the words each user finds hardest, in both Hebrew and English. Available on both Google Play and the App Store. אפליקציית מובייל ללימוד אוצר מילים לתלמידים המתכוננים לפסיכומטרי. נבנתה לגמרי לבד, מה-Pipeline של הנתונים וסיווג הקושי ב-ML ועד ה-UI ב-Flutter ופרסום ב-Stores. מילומטרי משתמשת בלמידה אדפטיבית כדי להציג לכל משתמש את המילים שהוא מתקשה בהן ביותר, בעברית ובאנגלית. זמינה ב-Google Play ובאפ Store.

What I Builtמה בניתי

Complete Flutter app with clean state management, local data persistence, and a smooth card-based review UI. Supports two word banks (Hebrew + English) with thousands of vocabulary items. אפליקציית Flutter מלאה עם ניהול State נקי, שמירת נתונים מקומית וממשק כרטיסיות חלק לחזרה על מילים. תומכת בשני בנקי מילים (עברית + אנגלית) עם אלפי פריטי אוצר מילים.

Data PipelinePipeline נתונים

Wrote Python scripts (classify_levels.py, rate_difficulty.py) to automatically classify and rank vocabulary words by difficulty using frequency analysis and ML techniques. כתבתי סקריפטים ב-Python (classify_levels.py, rate_difficulty.py) לסיווג ודירוג אוטומטי של מילות אוצר לפי רמת קושי בעזרת ניתוח תדירות וטכניקות ML.

Architectureארכיטקטורה

Clean architecture with separated layers for data, domain, and presentation. Optimised for fast startup and smooth performance on low-end Android devices. ארכיטקטורה נקייה עם שכבות מופרדות לנתונים, Domain ו-Presentation. אופטימיזציה לזמן הפעלה מהיר וביצועים חלקים גם על מכשירי Android חלשים.

Deploymentפרסום

Published to both Google Play and the App Store using Codemagic CI/CD. Maintained a privacy policy page and handled full store compliance end-to-end. פורסם ב-Google Play ובאפ סטור בעזרת Codemagic CI/CD. טיפול מלא בדרישות ה-Store כולל דף מדיניות פרטיות.

KJ Fitness App

KJ

KJ Fitness — Coaching Mobile App KJ Fitness — אפליקציית אימון מובייל

React Native Expo Firebase Firestore iOS & Android

A full fitness coaching platform built for a real client — KJ Fitness. The app serves two roles: clients see their weekly training plans, track sessions, and pay for sessions; coaches manage client rosters, create weekly content, and monitor progress. Built with React Native + Expo for cross-platform delivery and Firebase for real-time backend. פלטפורמת אימון מלאה שנבנתה עבור לקוח אמיתי — KJ Fitness. האפליקציה משרתת שני תפקידים: לקוחות רואים תוכניות אימון שבועיות, עוקבים אחרי אימונים ומשלמים; מאמנים מנהלים לקוחות, יוצרים תוכן שבועי ועוקבים אחרי התקדמות. נבנה עם React Native + Expo לפיתוח Cross-platform ו-Firebase ל-Backend בזמן אמת.

Dual-role Architectureארכיטקטורת שני תפקידים

Separate screen flows for clients and coaches, unified under a single codebase. Role-based access controlled via Firebase Auth and Firestore security rules. זרימות מסך נפרדות ללקוחות ולמאמנים, תחת Codebase אחד. בקרת גישה מבוססת תפקיד דרך Firebase Auth וחוקי אבטחת Firestore.

Client Featuresפיצ'רים ללקוח

Weekly plan cards, session tracking, in-app payment (Bit Pay integration), quick-action grid, and bottom tab navigation with 5 tabs. כרטיסיות תוכנית שבועית, מעקב אימונים, תשלום In-app (Bit Pay), רשת פעולות מהירות וניווט Tab תחתון עם 5 טאבים.

Tech StackStack טכנולוגי

React Navigation v6, Firebase Firestore + Auth + Storage, Expo managed workflow, custom theme system with design tokens (colors, typography, spacing). React Navigation v6, Firebase Firestore + Auth + Storage, Expo managed workflow, מערכת Theme מותאמת עם Design Tokens (צבעים, טיפוגרפיה, ריווח).

Backend

Firestore with complex security rules ensuring clients only access their own data. Custom indexes for efficient coach-client queries. Firebase Storage for profile media. Firestore עם חוקי אבטחה מורכבים שמבטיחים שלקוחות ניגשים רק לנתונים שלהם. Indexes מותאמים לשאילתות coach-client יעילות. Firebase Storage לתמונות פרופיל.

KJ Fitness Website

KJ Fitness — Coaching Website KJ Fitness — אתר האימון

HTML / CSS / JS Firebase Auth Firestore Responsive

The web companion to the KJ Fitness mobile app. A full multi-page coaching website that includes a marketing landing page, a Firebase-authenticated login system, and separate dashboards for clients and coaches. Clients view their plans and progress; coaches manage clients and edit weekly content — all without a traditional backend server. האתר הנלווה לאפליקציית KJ Fitness למובייל. אתר אימון רב-עמודי מלא הכולל דף נחיתה שיווקי, מערכת התחברות עם Firebase ודשבורדים נפרדים ללקוחות ולמאמנים. לקוחות צופים בתוכניות ובהתקדמות; מאמנים מנהלים לקוחות ועורכים תוכן שבועי — הכל ללא שרת Backend מסורתי.

Pages Builtעמודים שנבנו

Landing page (index.html), login page, client dashboard, and coach dashboard — each a fully functional HTML/CSS/JS page. דף נחיתה (index.html), דף התחברות, דשבורד לקוח ודשבורד מאמן — כל אחד עמוד HTML/CSS/JS פונקציונלי לחלוטין.

Auth & Dataאימות ונתונים

Firebase Auth for secure login. Firestore for reading and writing client/coach data live in the browser — no backend server required. Firebase Auth להתחברות מאובטחת. Firestore לקריאה וכתיבה של נתוני לקוח/מאמן חיים בדפדפן — ללא שרת Backend נדרש.

Designעיצוב

Fully responsive layout. Dark theme with teal accents matching the mobile app's brand. Optimised for both desktop and mobile browsers. פריסה Responsive מלאה. ערכת צבעים כהה עם הדגשות טורקיז התואמת לעיצוב האפליקציה. מותאמת גם לדסקטופ וגם לנייד.

Real Client Deliveryמסירה ללקוח אמיתי

Designed and delivered for KJ Fitness — an actual coaching business. Handled client requirements, branding integration, and production deployment. עוצב ונמסר ל-KJ Fitness — עסק אימון פעיל. טיפלתי בדרישות הלקוח, שילוב המיתוג ופריסה לפרודקשן.
