Untitled 3D PVP Multiplayer Game

Unity C# Multiplayer PVP 3D

A competitive 3D PVP game built in Unity. The core loop is fast-paced player-versus-player combat in a 3D environment, focused on skill expression and replayability. Multiplayer networking is a central challenge — designing for low-latency feel, fair matchmaking, and a responsive combat system that works well across network conditions.

This project is an opportunity to go deeper into Unity's architecture, multiplayer frameworks, and 3D game feel design — areas I'm actively building expertise in.

  • Real-time 3D PVP combat
  • Networked multiplayer
  • Custom character controller
  • Matchmaking system
  • Leaderboards & progression
  • Cross-platform (PC & mobile)
