About Me אודות

Building things that matter בונה דברים שחשובים

I'm a self-driven developer who ships mobile apps and web products that real people use. I built and released Milometri — a vocabulary learning app for Israel's psychometric exam — solo, end to end: Python data pipeline, Flutter UI, and App Store deployment.

אני מפתח עצמאי שמפרסם אפליקציות מובייל ומוצרי ווב שאנשים אמיתיים משתמשים בהם. פיתחתי ושחררתי את מילומטרי — אפליקציית לימוד אוצר מילים לפסיכומטרי — לבד, מהתחלה ועד הסוף: Python data pipeline, Flutter UI, ופרסום ב-App Store.

I also delivered the full KJ Fitness platform for a real client: a React Native coaching app backed by Firebase, plus a multi-page website with client and coach dashboards. I take complete ownership, from idea to production.

סיפקתי גם את פלטפורמת KJ Fitness המלאה ללקוח אמיתי: אפליקציית אימון ב-React Native מגובה ב-Firebase, ואתר רב-עמודי עם דשבורדים ללקוחות ולמאמנים. אני לוקח אחריות מלאה, מהרעיון ועד הפרודקשן.

I work across the full stack — mobile, web, and backend — and I care deeply about clean architecture, smooth UX, and code that holds up over time. Currently looking for a high-tech opportunity where I can contribute and grow.

אני עובד על פני כל ה-Stack — מובייל, ווב ו-Backend — ואכפת לי מאוד מארכיטקטורה נקייה, UX חלק וקוד שמחזיק לאורך זמן. מחפש כעת הזדמנות בהייטק להתפתח ולתרום.
